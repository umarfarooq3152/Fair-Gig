# Frontend (Next.js 15)

Role-based FairGig frontend for workers, verifiers, and advocates.

Install command:
`npm install`

Start command:
`npm run dev`

Environment variables:
- `NEXT_PUBLIC_AUTH_SERVICE_URL=http://localhost:8001`
- `NEXT_PUBLIC_EARNINGS_SERVICE_URL=http://localhost:8002`
- `NEXT_PUBLIC_ANOMALY_SERVICE_URL=http://localhost:8003`
- `NEXT_PUBLIC_GRIEVANCE_SERVICE_URL=http://localhost:8004`
- `NEXT_PUBLIC_ANALYTICS_SERVICE_URL=http://localhost:8005`

Routes:
- `/login`
- `/register`
- `/dashboard`
- `/shifts`
- `/shifts/new`
- `/certificate`
- `/verifier/queue`
- `/advocate/analytics`
- `/advocate/grievances`
- `/community`
